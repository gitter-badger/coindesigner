Loading and manipulating external files

In this tutorial we'll cover the following topics:
  • Importing external files
  • Using manips to transform models in a scene
  • Using manips to control the lighting in a scene

Step by step procedure

  1. We start by selecting option Import Geometry from the Scene menu and loading the file cow.off
    2.  
  2. As we can see, coindesigner loads the file's content in a new SoSeparator.
    3.  
  3. Imported files can be modified in the same way as all other inventor nodes. Here, for example, we've added a SoTexture2 node, applying the texture cow_skin.png to the imported geometry.
    4.  
  4. Now we use the same method to import the file bunny.xyz. We can see this object is much larger than the cow, so we'll try to reduce it in size.
    5.  
  5. We insert a SoTabBoxManip at the start of the second file's SoSeparator.
    6.  
  6. We reduce it in size (using the corner "handles" of the SoTabBoxManip), and move the rabbit to a new position (using the side "handles" of the SoTabBoxManip), until we've got our objects placed where we want them.
    7.  
  7. Now we right-click with the mouse on SoTabBoxManip and select the option Convert to SoTransform.
    8.  
  8. As we see, the manip disappears, leaving intact the transformations we've applied to the rabbit.
    9.  
  9. To illuminate our scene we add a SoDirectionalLightManip at the start of the scene, modifying the direction and color of the light to suit our taste.
    10.  
  10. We can convert the soDirectionalLightManip to a SiDirectionalLight by the same method we used before. We observe that the red light now only focuses on our objects from above.
